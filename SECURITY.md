Security Considerations for COREX

· Transformation parameters bounded
· Autoencoder latent dimensions capped
· Noise injection SNR controlled

· Environment clustering with privacy-preserving aggregation
· No cross-environment data contamination

Intervention Consistency Engine

· Propensity score matching with caliper bounds
· Synthetic counterfactual generation with noise calibration
