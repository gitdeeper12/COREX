# COREX Architecture

## Pipeline Overview

COREX implements a four-module evaluation pipeline...

## Module Descriptions

1. **Statistical Stability**: Tests P(Y|X) invariance across subpopulations
2. **Representation Invariance**: Tests robustness under feature transformations
3. **Intervention Consistency**: Tests causal effect direction & magnitude
4. **Domain Robustness**: Tests cross-environment generalization
